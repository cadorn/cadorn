## Perspective

I am fascinated by declarative composition and layered indirection for the purpose of binding components into systems.

Similar to how *kubernetes binds service*s and *terraform binds infrastructure* I am working on mechanisms to **bind JavaScript/TypeScript components into systems and applications**.

I dislike how current JavaScript codebases bind functionality and believe we can do better by structuring toolchains to link reusable components using configuration.

I have been working on how to compose systems from components at every layer of the vertical stack. A component based approach promises real advantages especially now that we have AI co-workers that need strict boundaries.
